Detailed Description

Vector-space basis way to represent a combination of rotations and translations on vectors and positions.

See Mathematics group description for conventions.

Vector-space used is R^3. This representation is a particular base from R^3 that is rotated and translated. Algebra traditionally only considers vectors but we will consider positions as well through homogeneous coordinates, which are vectors with a starting point. Thus, our coordinate system has an origin.

We call "absolute system" the coordinate system in which the basis vectors of this particular coordinate system are expressed.
