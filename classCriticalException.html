Detailed Description

Exception to be thrown by the CRITICAL macro.

It is not intended to be thrown by the user, even if he could in theory. The prefered way to use it is through the CRITICAL macro. It contains an error message, the name of the function that threw it, the name of the file that contains the throw statement and the line at which this throw statement can be found.
